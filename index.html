Router

Router merupakan sebuah perangkat sebagai penghubung antar dua atau lebih jaringan untuk meneruskan data dari satu jaringan ke jaringan lainnya. Router berbeda dengan switch. Switch merupakan penghubung beberapa alat
untuk membentuk suatu Local Area Network (LAN). Router sangat banyak digunakan dalam jaringan berbasis teknologi protokol TCP/IP, dan router jenisa itu disebut juga dengan IP Router.

Berikut adalah tipe-tipe router:

  1. Router Statis

    2. Cara kerja router statis dapat dibagi menjadi beberapa bagian, yaitu:

    • Administrator jaringannya yang mengkonfigurasi router.
    • Router melakukan Router berdasarkan informai dalam table Router.

  2. Router Dinamis

    3. Router protocol berbeda dengan routed protocol. Router protocol adalah komunikasi antara router-router. Router protocol mengizinkan router-router untuk sharing informasi tentang jaringan dan koneksi antar router. Router menggunakan informasi ini untuk membangun dan memperbaiki table router nya. Contoh router protocol yaitu:

    • Router Informastion Protocol (RIP)
    • Interior Gateway Routing Protocol (IGRP)
    • Enchanced Interior Gateway Routing Protocol (EIGRP)
    • Open Shortest Path First

Referensi: BAB, I. "KOMUNIKASI DATA DAN JARINGAN KOMPUTER PADA MATERI ROUTER."
